# notes-app

This is a Personal notes app where you can see and mange your notes.

This app includes:
A register page where you signup with your email and password.
A login page, where you signin to get access.
A dashboard where you can see your notes,manage them and delete or edit them.

Technologies used:

Frontend:
React
Tailwind CSS for styling

 Backend:
Node.js and Express.js for the server,which will handle all the requests
Databases & Authentication will use Supabase to store all user data such as Notes and Login details.
